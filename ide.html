P3 IDE



Along with two teammates I helped design and build an in browser front-end web development enviroment.

My team wanted to come up with a rapid prototyping tool that could be used to quickly create and share front end website design.

We used React to handle the live updating and rendering of the text in the code editor panels. We used MongoDB to store our user's projects and allow the users to build a profile with us.
